5 used R-packages for xMAPr

  • R: A Language and Environment for Statistical Computing (R Core Team (2022))
  • bookdown: Authoring Books and Technical Documents with R Markdown (Xie (2023a))
  • rmarkdown: Dynamic Documents for R (Allaire et al. (2023))
  • data.table: Extension of data.frame (Dowle and Srinivasan (2023))
  • DT: A Wrapper of the JavaScript Library ‘DataTables’ (Xie, Cheng, and Tan (2023))
  • factoextra: Extract and Visualize the Results of Multivariate Data Analyses (Kassambara and Mundt (2020))
  • FactoMineR: Multivariate Exploratory Data Analysis and Data Mining (Husson et al. (2022))
  • ggrepel: Automatically Position Non-Overlapping Text Labels with ‘ggplot2’ (Slowikowski (2023))
  • ggsignif: Significance Brackets for ‘ggplot2’ (Ahlmann-Eltze and Patil (2022))
  • gridExtra: Miscellaneous Functions for “Grid” Graphics (Auguie (2017))
  • heatmaply: Interactive Cluster Heat Maps Using ‘plotly’ (Galili and O’Callaghan (2023))
  • knitr: A General-Purpose Package for Dynamic Report Generation in R (Xie (2023b))
  • magrittr: A Forward-Pipe Operator for R (Bache and Wickham (2022))
  • minpack.lm: R Interface to the Levenberg-Marquardt Nonlinear Least-Squares Algorithm Found in MINPACK, Plus Support for Bounds (Elzhov et al. (2023))
  • plotly: Create Interactive Web Graphics via ‘plotly.js’ (Sievert et al. (2022))
  • RColorBrewer: ColorBrewer Palettes (Neuwirth (2022))
  • reshape: Flexibly Reshape Data ((R-reshape?))
  • reshape2: Flexibly Reshape Data: A Reboot of the Reshape Package ((R-reshape2?))
  • rlang: Functions for Base Types and Core R and ‘Tidyverse’ Features (Henry and Wickham (2023))
  • scales: Scale Functions for Visualization (Wickham and Seidel (2022))
  • shiny: Web Application Framework for R (Chang et al. (2022))
  • shinyalert: Easily Create Pretty Popup Messages (Modals) in ‘Shiny’ (Attali and Edwards (2021))
  • shinyBS: Twitter Bootstrap Components for Shiny (Bailey (2022))
  • shinydashboard: Create Dashboards with ‘Shiny’ (Chang and Borges Ribeiro (2021))
  • shinyFiles: A Server-Side File System Viewer for Shiny (Pedersen et al. (2022))
  • shinyjs: Easily Improve the User Experience of Your Shiny Apps in Seconds (Attali (2021))
  • shinyWidgets: Custom Inputs Widgets for Shiny (Perrier, Meyer, and Granjon (2023))
  • tidyverse: Easily Install and Load the ‘Tidyverse’ (Wickham (2023))
